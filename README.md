# My Pygame
## Before Playing
`pip3 install -r requirements.txt`

## How to Play 
`python3 main.py`

### Game Rule
- Using *Arrow keys* to controll.
- Eating **meats** gets scores.
- Touching **bombs** reduces scores.
    -  Once your scores are less than 0, you lose.
- Pressing *q* at any time to quit the game
